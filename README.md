## Getting Started

* git clone https://github.com/ngoxuansy30082002/chatapp_firebase.git
* open terminal
* configuration your Firebase
* flutter pub get
* flutter run

## Authentication used in this App

- Email/Password

# Feature

* User Management
  - Update information
  - Register
  - LogIn
  - LogOut
* Chat Group Management
  - Create a new group
  - Join a group
  - Leave a group
  - View group members
  - Search for groups
* Chat
  - Send text messages
  - Send image
